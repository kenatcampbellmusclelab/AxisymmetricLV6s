# AxisymmetricLV6s
Fast computational simulation of cardiovascular system, with axisymmetric left ventricle

For details, see:
Moulton, M.J. and Secomb, T.W.  A fast computational model for circulatory dynamics: Effects of left ventricle-aorta coupling. Biomechanics and Modeling in Mechanobiology 22:947-959 (2023).
